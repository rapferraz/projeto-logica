<h1 align="left">Projeto - Aula Lógica</h1>

###

<p align="left">Neste projeto, realizamos um sistema de cadastro para atendimento de clientes de uma clínica odontológica, onde o usuário poderá verificar todo o seus dados e históricos de atendimentos.</p>

###

<h2 align="left">O projeto conta com:</h2>

###

<p align="left">✅ Tela de Login<br>✅ Tela de Cadastro<br>✅Dashboard com todas as informações do usuário <br>✅ Histórico de atendimentos cadastrados do usuário</p>
